# 制作の書斎（Phase 0 プロトタイプ）

ショー・演出リファレンスを、制作の問い・場面案・ビジュアル試作へつなぐ個人用の作業空間。

- メイン: **資料棚** — 作品データベースの閲覧（一覧・検索・詳細＋関連作品/関連表現の自動集約）
- サブ: **制作机**（場面スタディ・ビジュアル比較）／**舞台スケッチ**（コマ配置・背景描画・PNG書き出し）／**種火**（0→1の一行場面を配る）／**場面問答**（判断のエクササイズ）

## 使い方

`index.html` をブラウザで開くだけ（静的・依存なし）。

## データ

- `db.js` は正本データ（`../show-reference/data/*.json`、リポジトリ外）から生成した読み取り専用の索引
- 正本更新後の再生成: `python3 build_db.py`
- コード変更時は `index.html` 内の `?v=` を上げる（キャッシュ対策）

## 状態
